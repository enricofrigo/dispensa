## 🧰 Funzionalità principali

- 📝 Aggiungi, modifica ed elimina articoli nella tua dispensa (usa Open Food Facts per recuperare le informazioni)
- 🔔 Ricevi notifiche per prodotti in scadenza  
- 📦 Location personalizzabili
